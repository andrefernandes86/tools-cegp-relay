COMMON ISSUES & SOLUTIONS
═══════════════════════════════════════════════════════════════════════════════

Issue: "fatal: origin already exists"
Solution: git remote remove origin
          Then: git remote add origin https://github.com/...

Issue: "Permission denied (publickey)"
Solution: 1. Generate SSH key: ssh-keygen -t ed25519 -C "your@email.com"
          2. Add to GitHub: https://github.com/settings/keys
          3. Use SSH URL instead of HTTPS

Issue: "fatal: could not read Username"
Solution: Use personal access token or SSH key
          Or: git remote set-url origin https://YOUR_USERNAME:TOKEN@github.com/...

Issue: Files not showing on GitHub
Solution: Check .gitignore - make sure it's not excluding important files
          Run: git status (before commit to verify what will be pushed)
